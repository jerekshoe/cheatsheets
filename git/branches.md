# Branches
## Create
```sh
$ git checkout -b <branchName>
```
## Delete
#### local
```sh
$ git branch -D <branchName>
```
#### remote
```sh
$ git push origin --delete <branchName>
```
## Switch
```sh
$ git checkout <branchName>
```
## View
#### local branches
```sh
$ git branch
```
#### all
```sh
$ git branch -a
```
